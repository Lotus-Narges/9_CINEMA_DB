## DATEDIFF() Function:
The DATEDIFF() function compares two dates and returns the difference.
EX -> SELECT DATEDIFF(month, '2011-03-07' , '2021-06-24');

Utilisez DATEDIFF dans les clauses SELECT <list>, WHERE, HAVING, GROUP BY et ORDER BY.
